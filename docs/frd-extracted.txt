B1.  User  Onboarding  &  Profile  Creation  
ABSOLUTE  RULE  —  NON-NEGOTIABLE A  user  CANNOT  view  the  event  feed,  browse  attendees,  register  for  any  event,  send  any  message,  or  access  any  feature  of  the  app  until  their  profile  is  100%  complete  with  verified  photo,  required  bio,  and  minimum  interest  tags  selected.  This  is  the  single  most  important  differentiator  from  every  competing  platform.  The  profile  IS  the  product.  
 
B1.1  Account  Creation  Registration  options  —  user  may  choose  any  of  the  following:  •  Email  and  password  •  Apple  ID  (Sign  in  with  Apple)  •  Google  account  •  Instagram  login

  Age  confirmation  flow:  •  User  must  confirm  they  are  18  or  older  before  proceeding  •  Date  of  birth  entry  required  —  system  calculates  age  •  If  under  18:  access  denied,  appropriate  message  displayed  •  21+  age  gating  is  applied  per  individual  event  at  registration  —  not  at  account  level
